<h3 align="center">Automatically build Xmas cards with user inputs </h3>

```
pip install Pillow
```
<h3> Available Options </h3>
<ul>
<li>Text Color</li>
<li>Canvas Color</li>
<li>Canvas Border Color</li>
<li>Receiver Name</li>
<li>Author Name</li>
<li>Header Icons</li>
<li>Footer Icons</li>
<li>File name </li>
</ul>

<h3> Header Icon Options </h3>
